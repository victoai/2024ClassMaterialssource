컴포넌트에 값 전달하기
: 컴토넌트가 다른컴포넌트에 값을 전달해야 하는 상황이 발생함
: 컴포넌트 간에 값을 주고 받는 방법

Props란? (Properties)

리액트에서는 
부모가 -> 자식컴포넌트에 단일 객체 형태로 값을 전달 할 수 있다


Props로 값전달하기
주의사항: 부모만이 자식 컴포넌트에 전달 가능함


function App = ()=>{
    const name="우주연";

    return(
      <div className="App">
          <Header/>
          <Body  name={name} />
          <Footer/>
      </div>
    );
     
}
